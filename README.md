1. 目录说明  
  * /grunt 为各种grunt插件用到的文件目录  
  * /mock  为请求的mock数据目录  
  * /node_modules node模块目录  
  * /nodeUitl 一些node写的的小工具  
  * /src 真正的代码目录  
  * /src/common 放全局公用代码  
  * /src/mapp 销售线索手机端代码  
  * /src/module 业务逻辑代码  
  * /src/resource 各种引用资源 图片、css等
